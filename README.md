## Chủ đề: Xây dựng phòng chat online.

## Thông tin nhóm:
  - Nguyễn Văn Hiếu
  - Nguyễn Phương Anh
  - Bùi Tuấn Nhật

## Ngôn ngữ sử dụng: 
  - front_end: html, css, javascript.
  - back_end: python

## Các chức năng chính:
  - tham gia phòng chat
  - gửi tin nhắn văn bản
  - gửi tin nhắn hình ảnh
  - gửi tin nhắn thoại

## thư viện sử dụng: flask, socketio, opencv.
